# Happy Women's Day — Prerana 💕

A special static website to wish Women's Day to Prerana, with a pink theme, sweet messages, and a **pink flower rain** when you click any button.

## What's inside

- **index.html** — Main page with personalized content for Prerana (born May 19, 2005)
- **styles.css** — Pink & gold theme, animations, responsive layout
- **script.js** — Flower rain effect on button clicks (no dependencies)

## Run locally

Open `index.html` in a browser, or use a simple server:

```bash
# Python
python -m http.server 8000

# Node (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Deploy on GitHub

1. Create a new repo (e.g. `womens-day-prerana`).
2. Push this folder:
   ```bash
   cd womens-day-prerana
   git init
   git add .
   git commit -m "Happy Women's Day, Prerana"
   git remote add origin https://github.com/YOUR_USERNAME/womens-day-prerana.git
   git push -u origin main
   ```
3. Go to **Settings → Pages**.
4. Source: **Deploy from a branch**.
5. Branch: **main**, folder: **/ (root)**.
6. Save. Your site will be at `https://YOUR_USERNAME.github.io/womens-day-prerana/`.
